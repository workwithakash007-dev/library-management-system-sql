# 📚 Library Management System SQL Project

## Overview

The Library Management System is a SQL-based database project that simulates the operations of a modern library. It is designed to manage books, members, employees, branches, book issuance, and return records efficiently. The project demonstrates database design principles, relational modeling, and advanced SQL querying techniques using a real-world business scenario.

---

## Objectives

* Design a structured relational database for library operations.
* Manage books, members, and employees effectively.
* Track book issuance and return transactions.
* Generate meaningful reports using SQL queries.
* Practice real-world SQL concepts used in Data Analytics and Database Management.

---

## Database Schema

The project consists of the following tables:

### Branch

Stores information about library branches.

### Employee

Maintains employee and manager details.

### Members

Contains registered library member information.

### Books

Stores book inventory and availability details.

### Issued_Status

Tracks books issued to members.

### Return_Status

Tracks returned books and return dates.
